# Link Title Maker

Turn any web address into a clean, clickable word or emoji. Copy the result as HTML (for email and documents) or as Markdown.

**Use it here:** https://pyender.github.io/link-title-maker-v3-1/

Free to use, runs entirely in your browser, needs no account, and is available in twelve languages. Current version: v3.5.2.

## What it does

Link Title Maker converts long, unwieldy URLs into a short clickable title or emoji, so a message or document can show a tidy label instead of a raw address. The result can be copied as HTML for email and documents, or as Markdown.

## Quick steps

1. **Prefix (optional):** type plain text that appears before the link (for example, `view:`).
2. **Link text / Emoji:** type any text, or click an emoji button.
3. **URL:** paste the full web address (it must start with `https://` or `http://`).
4. **Choose mode:**
   - **Link: text/emoji only** – only the word or emoji is clickable.
   - **Link: Prefix + text/emoji** – the entire phrase is clickable.
5. Click **Generate**, then **Copy for email/docs (HTML)** or **Copy Markdown**.

## Notes

- Some chat apps (for example, WhatsApp) do not support custom HTML links. That is a platform limitation, not a fault of the tool.
- Your language preference is stored in your browser's localStorage (`ltm_lang`) only.
- The live site uses a privacy-friendly visit counter (GoatCounter): it records
  anonymous visit counts only – no cookies, and no tracking of individuals.
